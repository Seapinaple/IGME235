Project 2 Documentation

Description

A web app using the scryfall api that allows you to post decklists in the search box and receive either a list of all the cards, a way to check legality of all the cards, or view them formatted in a way such that they can be prints out and used as proxies.

Requirements

  • Used the scryfall API

  • 3 different search functions include legality checking, print formatting, and a search box

  • Code does not crash

  • Uses local storage to save last search term and to move the images to the print formating page
